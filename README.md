# Electverse
ElectVerse — An immersive web experience that breaks down every major election system worldwide through cinematic visuals, live simulators, and AI-powered guidance. Built for students, journalists, and curious citizens everywhere. 🌐 Available in 10 languages — English, Hindi, French, German, Spanish, and more. Democracy deserves to be understood.
How to Run
bashgit clone https://github.com/yourname/electverse
cd electverse
npm install
npm run dev

Required API Keys
Create a .env file in root:
VITE_ANTHROPIC_API_KEY=your_key_here
VITE_GNEWS_API_KEY=your_key_here

Anthropic — console.anthropic.com
GNews — gnews.io



Built with React · Framer Motion · D3.js · Pollinations.ai
